# Getting Started
1. Clone the repo 
```git clone https://github.com/Afterdie/Oratory.git```

## Install dependencies

The project using npm
```npm install```

## Start dev server

```npm run dev```

Visit [localhost:3000](http://localhost:3000) to see the website

## Create Build

```npm run build```
